# Report Design Contract

## Audience

Technical reviewer evaluating model methodology, implementation quality, reproducibility, and claim safety.

## Reading path

Answer first, then model-vs-baseline evidence, feature evidence, data scope, methodology, limitations, and next steps.

## Visual contract

- Use grouped bars for same-unit classifier metrics across model and baseline.
- Use sorted horizontal bars for permutation importance.
- Use at most two restrained non-neutral palette roots plus neutral text and grid lines.
- Keep all rates on a 0–1 scale and feature importance as held-out F1 decrease.
- Pair every chart with an adjacent interpretation and limitation.

## Accessibility and portability

Use the packaged Data Analytics report reader, semantic fallbacks, responsive single-column reading flow, and canonical source affordances. Do not require network access.
